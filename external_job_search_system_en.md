# External Job Search System (EN Global)

## Target Roles
- Product Analyst (Games)
- Game Analyst / Live Ops Analyst
- AI Operations Analyst / AI Integrator (Product Ops context)

## Weekly Output Targets
- 5-7 targeted applications.
- 2 tailored resume variants (if needed by role family).
- 2 portfolio-link refreshes based on rejection/interview feedback.

## Job Fit Checklist
Use only roles where at least 6/10 are true:
- Player behavior / retention / product metric ownership.
- Cross-functional work with product/design/engineering.
- Emphasis on data-informed prioritization.
- Comfortable with qualitative + quantitative signal blending.
- Entry path for transition from operations/support.

## Application Message Structure
1. Hook: current game/support domain exposure.
2. Proof: one relevant case with measurable structure.
3. Fit: role requirement match in 2-3 bullets.
4. CTA: request intro call.

## Conversion Metrics
- Application -> Recruiter response rate.
- Recruiter response -> Interview rate.
- Interview -> Next-stage rate.

## 4-Week Recovery Rule (If Conversion Is Low)
If no interviews in 4 weeks:
1. Rebuild headline + summary in CV.
2. Replace weakest portfolio case with stronger narrative framing.
3. Match keywords exactly to top 20 target JD terms.
4. Increase referral-based applications share.
